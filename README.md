## Herokuへのデプロイ手順
- パス
  - https://whispering-chamber-06924.herokuapp.com
### Herokuへ直接デブロイする方法
1. git push heroku 開発ブランチ名:master
1. heroku run rails db:migrate
### Herokuへの自動デプロイ手順に関して
1. masterにMergeされた時、テスト項目が全てOKなら自動的にデプロイされます。
